### 👋 Chào bạn

Mình là một dev tự học, làm chủ yếu một mình từ ý tưởng đến lúc chạy thật.
Quan tâm tới **web app, tự động hoá, và tích hợp AI** — thích những thứ nhỏ gọn,
chạy ổn định và giải quyết được một vấn đề cụ thể. Còn nhiều thứ phải học, nên rất
hoan nghênh góp ý.

### 🛠 Mình hay làm việc với

- **Web**: Next.js / React, Node.js, FastAPI, TypeScript, Python
- **AI**: tích hợp LLM (Claude · Gemini · Ollama self-host), RAG cho dữ liệu riêng
- **Hạ tầng**: Docker, Cloudflare Workers/Pages, PostgreSQL, Linux VPS
- **Khác**: bot Telegram, xử lý ảnh, dashboard real-time

### 🌱 Đang tìm hiểu

Cloudflare Agents SDK, kiến trúc RAG gọn nhẹ, và viết code sạch hơn — dễ bảo trì hơn.
